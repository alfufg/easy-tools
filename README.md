# Easy Tools - 浏览器扩展

一个多工具合一的 Chrome/Edge 浏览器扩展，集成多种实用功能，帮助你高效采集数据和管理内容。

## 功能特性

- **小红书评论采集**: 支持按关键字、链接、达人 ID 采集评论数据
- **抖音评论采集**: 采集抖音视频评论，支持多种筛选条件
- **抖音视频下载**: 无水印视频下载助手
- **抖音视频录屏**: 自动录屏取证功能
- **星图视频采集**: 达人视频数据采集
- **SEO 排位查询**: 多平台搜索收录排位查询

## 技术栈

- Chrome Extension Manifest V3
- ES6+ JavaScript (模块化)
- IndexedDB 数据存储
- Layui UI 框架

## 安装方法

### 开发者模式安装

1. 下载本仓库代码
2. 打开 Chrome/Edge 浏览器，进入扩展管理页面
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录

### 商店安装

- Chrome Web Store: (上架中)

## 使用说明

1. 点击浏览器工具栏上的扩展图标
2. 在弹出的工具面板中选择需要的功能
3. 各工具都有独立的使用界面和说明

## 权限说明

本扩展需要以下权限来正常工作：

- `tabs`: 访问浏览器标签页
- `sidePanel`: 使用侧边栏面板
- `scripting`: 在网页中执行脚本
- `activeTab`: 访问当前活动标签页
- `webNavigation`: 监听页面导航事件
- `notifications`: 显示通知
- `downloads`: 下载文件
- `cookies`: 访问 Cookie（用于登录状态）
- `storage`: 本地数据存储

## 隐私政策

本扩展仅收集以下数据：

- 用户在扩展界面中输入的配置信息
- 采集的数据仅存储在本地浏览器中
- 不会上传任何数据到远程服务器
