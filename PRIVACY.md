# 隐私权政策 | Privacy Policy

**最后更新日期 / Last Updated:** 2024年4月1日

---

## 中文版

### 1. 信息收集

Easy Tools 浏览器扩展（以下简称"本扩展"）收集的信息包括：

- **用户设置数据**：用户在扩展界面中输入的配置信息（如导出格式偏好）
- **临时处理数据**：从小红书、抖音等平台导出的内容（仅在本地处理）
- **本地存储数据**：扩展设置和用户偏好

### 2. 数据存储

- 所有数据**仅存储在用户本地浏览器**中（使用 Chrome Storage API）
- **不会上传**任何数据到远程服务器
- 用户可以通过浏览器设置随时清除本地数据

### 3. 数据使用

本扩展使用收集的数据仅用于：

- 实现内容备份和导出功能
- 存储用户设置和偏好
- 改善用户体验

**我们承诺：**
- ❌ 不出售用户数据
- ❌ 不与第三方共享用户数据
- ❌ 不用于广告投放
- ❌ 不进行用户追踪

### 4. 数据访问权限

本扩展需要访问以下网站以实现核心功能：

- `*.xiaohongshu.com` - 小红书（用于导出用户自己的内容）
- `*.douyin.com` - 抖音（用于导出用户自己的内容）

**重要声明：**
- 仅访问用户主动打开的页面
- 仅导出用户自己拥有权限的内容
- 不会在后台自动访问任何网站

### 5. 第三方服务

本扩展使用以下本地第三方库（不涉及远程服务）：

- jQuery 3.7.1
- Layui
- SheetJS (xlsx)
- Handsontable
- Moment.js
- FileSaver.js

### 6. 用户权利

用户有权：

- 查看存储在本地的所有数据
- 通过浏览器设置清除扩展数据
- 卸载扩展以删除所有本地数据
- 拒绝授予主机权限（但将无法使用核心功能）

### 7. 儿童隐私

本扩展不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

### 8. 隐私政策更新

我们可能会不时更新本隐私政策。更新后，我们会在此页面发布新版本。

### 9. 联系我们

如有任何隐私相关问题，请联系：

- **邮箱**：support@example.com
- **GitHub Issues**：https://github.com/ben/easy-tools/issues

## English Version

### 1. Information Collection

Easy Tools browser extension ("the Extension") collects:

- **User Settings**: Configuration preferences entered in the extension interface
- **Temporary Processing Data**: Content exported from Xiaohongshu, Douyin, etc. (processed locally only)
- **Local Storage Data**: Extension settings and user preferences

### 2. Data Storage

- All data is **stored locally in the user's browser** (using Chrome Storage API)
- **No data is uploaded** to remote servers
- Users can clear local data anytime through browser settings

### 3. Data Usage

The Extension uses collected data solely for:

- Implementing content backup and export features
- Storing user settings and preferences
- Improving user experience

**We Promise:**
- ❌ We do NOT sell user data
- ❌ We do NOT share user data with third parties
- ❌ We do NOT use data for advertising
- ❌ We do NOT track users

### 4. Data Access Permissions

The Extension requires access to the following websites for core functionality:

- `*.xiaohongshu.com` - Xiaohongshu (for exporting user's own content)
- `*.douyin.com` - Douyin (for exporting user's own content)

**Important Notice:**
- Only accesses pages user actively opens
- Only exports content user has permission to access
- Does NOT automatically access any websites in the background

### 5. Third-Party Services

The Extension uses the following local third-party libraries (no remote services):

- jQuery 3.7.1
- Layui
- SheetJS (xlsx)
- Handsontable
- Moment.js
- FileSaver.js

### 6. User Rights

Users have the right to:

- View all locally stored data
- Clear extension data through browser settings
- Uninstall the extension to delete all local data
- Deny host permissions (but core features will be unavailable)

### 7. Children's Privacy

The Extension is not intended for children under 13. We do not knowingly collect personal information from children.

### 8. Privacy Policy Updates

We may update this privacy policy from time to time. Updates will be posted on this page.

### 9. Contact Us

For any privacy-related questions, please contact:

- **Email**: support@example.com
- **GitHub Issues**: https://github.com/ben/easy-tools/issues
